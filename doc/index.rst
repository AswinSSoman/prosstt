PROSSTT - a python package for simulation of single-cell RNA sequencing data
============================================================================

Welcome to PROSSTT's documentation! PROSSTT is a python package for the simulation of single-cell RNA sequencing data of differentiation processes. It can simulate arbitrary differentiation topologies that are recognized by state-of-the-art trajectory inference algorithms like Monocle, SLICER and destiny.

PROSSTT is free software distributed under the GNU General Public License (GPL), version 3.0 (see :ref:`license`)

The git repository is https://github.com/soedinglab/prosstt
